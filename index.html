A brief foray into the Sketchfab API for the mixing of 2 textures.
It uses a hidden html canvas and reduces the images on the fly when
the slider is moved to speed things up.

Press and hold (be patient) the big yellow button to load a
higher res texture for a lovely detailed view

Currently only works on models with one material, and not older models.
Styles, layout and functionality are in no way optomized
and it is a matter of blind faith that it will work for other people.

Additional comments in source, and all available on Github

* If trying it yourself locally you may need to disable 'local file restrictions'.
